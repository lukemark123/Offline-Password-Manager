# Offline Password Manager

A secure, offline password manager for Windows.

## Features
- Fully offline, local vault
- Encrypted with master password
- Auto-lock and clipboard auto-clear
- Password generator with strength and entropy
- Favorites, password history, offline audit
- Backup and restore locally

## Installation
Download the `.exe` and run it. No additional dependencies required.

## Usage
1. Set a strong master password (≥8 characters).  
2. Add your accounts, generate strong passwords.  
3. Use offline audit to detect weak/reused passwords.  
4. Vault auto-locks on inactivity; clipboard clears automatically.
